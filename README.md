# epsi-vue-npx

Vue project for framework js module.

This application is a pokedex application.

## Project Setup

```sh
# clone project
git clonehttps://github.com/QuentinVdr/epsi-vue-npx
# install dependencies
npm install
# run locally
npm run dev
```

## Project idea

Pokedex using [pokeapi](https://pokeapi.co/)
