# epsi-vue-npx

Vue project for framework js module

## Project Setup

clone project

```sh
git clonehttps://github.com/QuentinVdr/epsi-vue-npx
pnpm install
```

install dependencies

```sh
npm install
yarn install
pnpm install
```

run locally

```sh
npm run dev
yarn dev
pnpm dev
```
